## Feature
* 주변 환경에 대한 제약을 최소화한 실용적 Active Noise Cancelling 기능
* 부가적인 SW 프로그램 설치 없이 사용 가능한 인터페이스
* 다양한 Target Device에서 적용 가능한 인터페이스
* 여러 소프트웨어 모듈을 이용한 Noise Cancelling 기능 정확도 향상
